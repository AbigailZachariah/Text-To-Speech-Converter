# Text-to-Speech Converter

A Text-to-Speech Converter built using HTML, CSS, and JavaScript.

## Features

- Convert typed text into speech
- Clean user interface
- Smooth gradient background
- User-friendly controls

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Type

Front-End Web Development Project

This project is completely front-end based and does not require any backend or database.

## How It Works

1. User enters text into the input box
2. JavaScript captures the text
3. Browser Speech Synthesis API converts text into speech
4. Audio is played directly in the browser
